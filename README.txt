Собираем
  Simple Computer:
  1.Скачиваем репозиторий;
  2.Открываем терминал.
  3.Прописываем  make и жмем Enter.
  4.В терминале пишем simpleComputer.

  Simple Assembler Translator:
  1.Открываем терминал в директории sat.
  2.Так-же прописываем make и жмем Enter.



Чтобы транслировать код из Simple Assembler, запишите код в файл с расширением .sa.
Затем запустите в терминале sat source.sa source.o.
Запустите SimpleComputer и загрузите файл source.o(клавиша l -> source.o)
